# Dashboard de Vendas – Power BI

## Objetivo
Desenvolver um dashboard gerencial para análise de vendas, permitindo o acompanhamento de faturamento, desempenho por produto, região e período.

## Descrição do Projeto
Este projeto simula um cenário real de empresa, utilizando dados fictícios de vendas para criação de indicadores estratégicos e apoio à tomada de decisão.

## Principais Métricas
- Faturamento total
- Ticket médio
- Quantidade vendida
- Faturamento por região
- Evolução mensal das vendas

## Estrutura do Repositório
- dados: base de dados utilizada no projeto
- powerbi: arquivo do dashboard (.pbix)
- imagens: prints do dashboard

## Ferramentas Utilizadas
- Power BI
- Excel

## Habilidades Demonstradas
- Modelagem de dados
- Criação de métricas com DAX
- Visualização de dados
- Análise de indicadores de negócio
